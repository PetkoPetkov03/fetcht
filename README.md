# FETCHT

<p>Fetcht is a tool that helps me grab my github auth token from whatever directory i currently am in </p>

<i>FLAGS</i>

### -c == Changes the current token
### -h == Showcases the current working flags

## Usage

### fetcht -- get token
```
    fetcht
```

### fetch -- rewrite token
```
    fetcht -c ![#c5f015](token) `#c5f015`
```
